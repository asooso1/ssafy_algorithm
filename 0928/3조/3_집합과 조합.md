## 3 번

![집합과조합론_3](3_집합과 조합.assets/집합과조합론_3.PNG)

Ans)

원소의 개수가 0,1,2...k...n개일때의 부분집합의 수는 nC0,nC1,nC2,...nCk...nCn개이다.

위 식에 x = y = 1을 대입한다면

2^n = nC0 + nC1 + nC2 + ... + nCn이기에 증명할 수 있다.
