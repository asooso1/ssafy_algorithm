## 기초수식



### 2번 문제

```
T(n) = T(n-1) + n일때

= ...

T(n) = T(n-k) + kn + a (a는 정수)
k가 n일때 T(0) + N **2 + a  = n **2 + a +1.
따라서 시간복잡도는 O(n**2)
```



### 4번 문제

```
T(n) = T(n/2)+1
= T(n/4)+1 +1
= T(n/8)+1 + 1 + 1
...
= T(n/2^k) + k   

k가 logn일 때, T(1) + log n 
따라서 시간복잡도는 O(logn)
```



### 6번 문제

```
T(n) = 2(T(n/2) +n 일때

= ...

T(n) = (2^k)(T(n/2^k)+n/2^k) + kn

K가 log n 일때 (밑이 2)

T(n)= nT(1) + nlog(n) 

따라서 시간 복잡도는 따라서 O(nlog(n))
```
